== Description ==

Free question and answers section for each posts of your blog with neat and clean set-up.

== Installation ==

- Unpack zip file and load to wp-contents/plugins/ directory of your wordpress.
- After loading, login to your wordpress blog and go to Plugins section and then to 'iSnare Answers'
- Input the API key you created at https://www.isnare.org/api_widgets/
- Activate the plugin

That's it!
